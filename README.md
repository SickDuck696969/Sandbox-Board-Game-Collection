# Sandbox Board Game Collection

A lightweight collection of classic board games built using **Java Swing**. This project focuses on a "sandbox" approach where players manage the game flow locally while the UI handles physical components, calculations, and animations.

## 🎮 Included Games

### 1. Yahtzee
A fully digital score-sheet and dice-rolling tray.
* **Dice Animations:** Realistic rolling effects using timed transitions.
* **Hover Preview:** View potential scores in red before committing to a category.
* **Auto-Scoring:** Automatic calculation of Upper Total, Bonus, and Grand Total.
* **Seamless Controls:** Left-click to hold/score, Right-click to roll/undo.

### 2. Hive
A hexagonal tile-laying game featuring a free-form board.
* **Hexagonal Logic:** Custom precision hitboxes for accurate piece placement.
* **Beetle Stacking:** Managed Z-order allowing pieces to stack on top of each other.
* **Multi-Language Rules:** Built-in quick guides in both English and Vietnamese.

### 3. Tellstones
A minimal, strategic memory game.
* **Intuitive UI:** Simple drag-and-drop mechanics.
* **Image Support:** High-quality rendering of game tokens.

---

## 🛠 Features

* **Game Selector:** A central launcher to switch between games. The hub automatically hides when a game starts and reappears when it is closed.
* **Flat Design UI:** A modern dark-themed interface with high-contrast elements.
* **Randomized Aesthetics:** Dynamic color randomization for dice and UI components.
* **Local Multiplayer:** Designed for 2-4 players sharing the same screen.
